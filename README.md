# Terfon

**Terfon** é um jogo de adivinhação de palavras inspirado no popular *Wordle* (conhecido no Brasil como *Termo*), mas com um foco principal na **fonética**. Neste jogo imperdoável, os jogadores tentam descobrir a palavra diária utilizando exclusivamente o **Alfabeto Fonético Internacional (IPA)**.

## 🎯 Sobre o Projeto

O propósito do Terfon não é apenas entreter, mas familiarizar de maneira lúdica e prática os seus jogadores com transcrições e símbolos fonéticos, promovendo o raciocínio além da ortografia convencional. 

Diferente do teclado comum, este projeto usa um **teclado virtual customizado com fonemas**, permitindo ações avançadas e imersivas conforme os jogadores progridem nas tentativas — tudo com uma única chance por dia.

## ✨ Funcionalidades Principais

- **Teclado Fonético (IPA):** Botões virtuais substituem as letras convencionais do alfabeto por formas de pronúncia e fonemas do sistema IPA.
- **Interação Avançada de Fonemas:** Suporte integrado a ações de *long-press* (pressão longa) para revelar sons baseados em variações e alternâncias rápidas (ex: segurar `∫` para exibir `tʃ`).
- **Desafio e Bloqueio Diário:** Assim como os grandes clássicos do gênero, o Terfon preserva a exclusividade da experiência. Concluída a partida (com vitória ou derrota), uma tela de resultados detalhada é exibida e o jogador fica impedido de jogar novamente no mesmo dia, mantendo o progresso e o desafio restritos diários.
- **Design Moderno e Responsivo:** Otimizado com cores marcantes tanto para desktop quanto para navegadores em dispositivos móveis.

## 🚀 Tecnologias e Ferramentas

Projeto centralizado em ferramentas rápidas e de fácil manipulação front-end:
- **[Vite](https://vitejs.dev/)** - O *bundler* responsável pela estruturação do projeto ágil e rápido.
- **Node.js, HTML5 e Vanilla Javascript/CSS moderno**.

## 📦 Como rodar o projeto localmente

Para explorar o código no seu próprio ambiente, siga os passos abaixo:

1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/seu-usuario/Terfon.git
   ```

2. Acesse a pasta do projeto principal:
   ```bash
   cd Terfon/Terfon-Project
   ```

3. Instale todas as dependências do projeto por meio do seu gerenciador preferido:
   ```bash
   npm install
   ```

4. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. O projeto estará disponível, geralmente, em `http://localhost:5173`.
